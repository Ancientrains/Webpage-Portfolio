Projects
2026-03-01

SDSS Datathon — Team 44 places 1st. 144× baseline improvement on Toronto shelter pressure prediction.

Brief: forecast Toronto shelter capacity pressure. Built a HistGradientBoostingRegressor pipeline on a 2-year dataset. Engineered occupancy-free features to prevent target leakage, one-hot encoded 6 categorical dimensions, validated via 80/20 temporal split.

Result: 144× improvement over baseline MAE, RMSE of 7.08 × 10⁻⁴ on logit-transformed scores. Location identified as primary pressure driver. GTA geospatial heatmap and stakeholder report delivered.

2025-12-15

MelonBX deployed to production — Siamese CNN predicting watermelon sweetness from images, live on AWS EC2.

Multi-branch Siamese CNN (PyTorch) achieving ~80% reduction in RMSE and MAE vs. baseline. Dataset: 2,000+ samples from Ontario farms, each with a ground-truth Brix refractometer reading.

Infrastructure: AWS EC2/Nginx/Gunicorn, custom domain, Certbot SSL, CI/CD pipeline, S3 persistence via IAM Instance Roles. Frontend prototyped in Figma, built in HTML/CSS/JS.

2025-05-10

Country Predictor published — annotated comparison of OLS, gradient descent, and closed-form linear regression.

Single-file notebook implementing all three approaches from scratch with mathematical derivations alongside the code. Compares convergence behaviour, computational cost, and numerical stability. A reference for understanding what sits underneath sklearn.

Full workflow: variable selection, collinearity checks, residual vs. fitted plots, Q-Q plots, Cook's distance, VIFs, AIC/BIC stepwise selection, and cross-validation. Written in R; serves as a reference implementation for clinical observational data analysis.
