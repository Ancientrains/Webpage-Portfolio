# Personal Webpage

This folder contains the static site files for GitHub Pages.

## Files
- `module_updated.html` - Main source file for the site.
- `index.html` - Thin redirect so GitHub Pages loads the site by default.
- `assets/` - Images, icons, and other static assets.

## GitHub Pages Setup
1. Push this repository to GitHub.
2. In GitHub, go to `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select the branch you want (usually `main`) and set the folder to `/personal_webpage`.
5. Save and wait for the build to finish. Your site URL will appear there.

## Local Preview
Open `module_updated.html` directly in a browser, or use a simple local server.

## Notes
If you move the files to the repo root or `/docs`, update GitHub Pages to that folder. If you rename `module_updated.html`, also update `index.html`.
